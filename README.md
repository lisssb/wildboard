# Dashboard

## Install

Install 'postgresql' on your system.

Create database:

    sudo -u postgres createuser -P dashboard
    sudo -u postgres createdb dashboard -O dashboard

Use `123` as user password. 

## Run

    node ./

The server is at `http://127.0.0.1:3000/`
